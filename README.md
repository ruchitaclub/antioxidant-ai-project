# 🌿 AI-Based Antioxidant Activity Prediction System

## 🚀 Project Overview
This project predicts the antioxidant activity of herbal compounds using Machine Learning.  
It uses molecular features such as molecular weight, hydrogen donors, hydrogen acceptors, and LogP.

## 🎯 Features
- Predict antioxidant activity
- Interactive web interface
- Graph visualization
- Deployed using Streamlit

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
